Detailed Description

template<class Lcp, class Cst>
class sdsl::_lcp_support_tree< Lcp, Cst >

This class composes a virtual lcp array from a lcp arrays which is ordered in suffix array order (e.g. lcp_kurtz or lcp_uncompressed) and a compressed suffix tree like cst_sct3 or cst_sada. The time consumption of the []-operator depends on:

  • The time consumption of the tlcp_idx function of the CST
  • The access time to the suffix array ordered LCP array
