Detailed Description

template<class wt_type = int_vector<>>
class sdsl::rrr_vector< wt_type >

A bit vector which compresses the input with the method from Raman, Raman, and Rao.

This compact representation was presented by Rajeev Raman, V. Raman and S. Srinivasa Rao at SODA 2002 in the article: Succinct Indexable Dictionaries with Applications to representations of k-ary trees and multisets. This implementation fixes the block size to 15 and includes optimizations for bit vectors which contains runs of length >= 15 .

For a version with variable block size see sdsl::rrr_vector_var .
